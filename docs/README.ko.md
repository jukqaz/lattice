# Lattice 문서

[English](README.md) | 한국어 | [Repository README](../README.ko.md)

이 디렉터리는 사람이 읽는 공개 문서와 LLM 에이전트용 작업 지침을 분리합니다.
공개 문서가 우선입니다. Lattice를 처음 보거나 실제로 쓰는 사람은 아래 순서로
읽으면 됩니다.

## 추천 읽기 순서

1. [사용자 가이드](user/usage.ko.md): 설치, 첫 Codex 백업, 안전한 복원, 다른
   서비스 추가, Git sync를 다룹니다.
2. [제품 범위](product/mvp-scope.ko.md): Lattice가 하는 일, 하지 않는 일, 현재
   릴리스 범위를 정리합니다.
3. [변경 로그](../CHANGELOG.ko.md): 릴리스별 변경점과 업그레이드 시 확인할
   내용을 봅니다.
4. [Repository README](../README.ko.md): 빠른 명령 reference로 사용합니다.
